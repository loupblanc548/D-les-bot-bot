> Commandes de modération

- `/mod warn [@user]` — Avertir un membre
- `/mod mute [@user]` — Rendre muet (timeout long)
- `/mod unmute [@user]` — Retirer le timeout
- `/mod kick [@user]` — Expulser
- `/mod ban [@user]` — Bannir
- `/mod timeout [@user]` — Timeout court terme
- `/mod clear [nombre]` — Supprimer messages
- `/mod unlock` — Déverrouiller le salon
- `/mod purge [@user]` — Supprime messages d'un utilisateur
- `/mod history [@user]` — Historique des messages
- `/mod slowmode [durée]` — Slowmode du salon
- `/mod lock` — Verrouiller le salon
- `/mod softban [@user]` — Soft ban (ban+unban)
- `/mod tempban [@user]` — Ban temporaire
- `/mod purgeuser [@user]` — Supprime tous les messages d'un user
- `/mod snipe` — Dernier message supprimé
- `/mod report [@user]` — Signale un membre au staff
